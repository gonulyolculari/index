# Gönül Yolcuları - Final Site Paketi

Bu sürüm GitHub Pages için hazırlanmıştır.

## Sayfalar
- `index.html` → Anasayfa
- `mezarlik-projesi/index.html` → Mezarlık / türbe projesi sayfası

## Seçilen sistem
Yönetim paneli yerine JSON tabanlı veri sistemi seçildi. GitHub Pages ile daha uyumlu, daha güvenli, ücretsiz ve bu proje için daha hafiftir.

## Türbe ekleme
1. Görseli `assets/images/covers/` içine koy.
2. `data/turbeler.json` dosyasındaki `turbeler` listesine yeni kayıt ekle.

JSON dosyası doğrudan `file://` ile açılınca bazı tarayıcılarda çalışmayabilir. GitHub Pages üzerinde çalışır.
